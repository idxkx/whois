# Change: 域名查询界面实时进度与未注册导出

## Why
目前 Web 表单提交后只能等待最终结果，无法看到逐条查询的进度；同时“未注册”结果只能靠人工复制。运营希望能实时查看每条 whois 查询状态，并能一键导出未注册列表。

## What Changes
- 在 Web UI 中展示实时执行过程，逐条输出“正在查询/已注册/未注册”等状态，并显示已完成/总数。
- 将每个未注册的组合域名汇总到一个独立列表，支持在前端预览与筛选。
- 提供浏览器端“保存为 txt”下载能力，导出未注册列表供后续资产登记。
- 如果无法继续查询（如限频或错误）应及时在 UI 上提示并允许重试。

## Impact
- Affected specs: domain-query-ui
- Affected code: HTTP 服务（API/推送）、前端页面逻辑、可能的静态资源
